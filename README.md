## How to run locally

```
git clone https://github.com/eskinderg/eskinderg.github.io.git
```

```
cd eskinderg.github.io
```

```
npm install
```

```
npm run start
```

<p>Navigate to <a href="http://localhost:4200/" target="_blank">http://localhost:4200/</a></p>
